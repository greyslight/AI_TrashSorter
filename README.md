# AI_TrashSorter
This is my first creation, a discord bot. Its main purpose is to sort garbage and can recognize them with a built in AI image recognizer. Feel free to try in your projects!

## Features
* Can classify pictures of garbage
* Bot can give you informations to handle your trash
* Can do other tasks than classify garbage
